Visual Language — Saia da Rotina

Objetivo

Toda representação visual deve reforçar a sensação de que o relacionamento é uma história em constante construção.

Os elementos visuais não são decoração.

Eles comunicam significado.

⸻

Princípio

Sempre que possível, utilizar símbolos que representem experiências humanas.

Evitar elementos associados a sistemas corporativos.

⸻

Recomendações

Nunca parecer tarefas.

Devem parecer convites.

A representação visual deve sugerir oportunidade.

⸻

Cards

Cada cartão deve possuir apenas um foco visual.

Nunca competir com outros cartões.

Muito espaço em branco.

Poucos elementos.

⸻

Ilustrações

Quando existirem, devem transmitir:

acolhimento

leveza

cotidiano

romance

Nunca exagero.

Nunca caricatura.

⸻

Emoção predominante

Ao olhar para qualquer tela o usuário deve sentir:

“Essa história é nossa.”

Nunca:

“Preciso preencher informações.”

⸻

Consistência

Um mesmo conceito deve sempre utilizar a mesma representação visual em todo o aplicativo.

Exemplos:

Memória → 💌

Capítulo → 📖

Conquista → 🏆

Experiência → ✨

Conexão → 🤝

Essa consistência é mais importante do que variedade.
