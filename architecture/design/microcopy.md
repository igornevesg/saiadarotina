Microcopy — Saia da Rotina

Objetivo

Toda mensagem exibida pelo aplicativo deve fortalecer emocionalmente a experiência do casal.

As palavras utilizadas fazem parte do produto.

Nunca escrevemos apenas para informar.

Escrevemos para acolher.

⸻

Voz da marca

O aplicativo conversa como um amigo próximo.

É gentil.

É elegante.

É otimista.

É romântico sem exageros.

Jamais infantil.

Jamais artificial.

Jamais motivacional em excesso.

⸻

Tratamento

Sempre utilizar:

“vocês”

Nunca:

“usuário”

Nunca:

“cliente”

Nunca:

“casal” como forma de tratamento.

O casal é o protagonista.

⸻

Exemplos

Em vez de:

“Você possui 10 memórias.”

Preferir:

“Vocês já construíram 10 memórias juntos.”

⸻

Em vez de:

“Pontuação 82”

Preferir:

“A história de vocês demonstra uma conexão muito forte.”

⸻

Em vez de:

“Crie uma experiência.”

Preferir:

“Que tal escrever mais um capítulo dessa história?”

⸻

Em vez de:

“Nenhuma memória encontrada.”

Preferir:

“A história de vocês ainda está começando. O primeiro capítulo pode nascer hoje.”

⸻

Linguagem

Preferir palavras como:

história

momento

descoberta

lembrança

memória

capítulo

jornada

conexão

carinho

experiência

celebração

presença

⸻

Evitar:

score

meta

objetivo

performance

produtividade

ranking

missão

desempenho

⸻

Feedback positivo

Sempre celebrar pequenas conquistas.

Exemplos:

“Mais uma lembrança para guardar.”

“Vocês continuam escrevendo uma linda história.”

“Cada experiência fortalece a relação.”

⸻

Feedback negativo

Nunca utilizar linguagem de culpa.

Evitar:

“Vocês precisam…”

“Está faltando…”

“Seu relacionamento…”

Preferir:

“Talvez seja um bom momento para criar novas lembranças.”

“Que tal viver uma nova experiência juntos?”

⸻

Estados vazios

Todo Empty State deve transmitir oportunidade.

Nunca abandono.

Nunca erro.

Nunca ausência.

Sempre possibilidade.

⸻

Chamadas para ação

Preferir:

Descobrir

Celebrar

Registrar

Explorar

Relembrar

Continuar

Viver

Evitar:

Cadastrar

Adicionar

Criar registro

Salvar item

Enviar formulário

⸻

Emoção

Ao terminar qualquer tela o usuário deve sentir:

“Foi bom lembrar disso.”

Essa sensação vale mais do que qualquer métrica.
